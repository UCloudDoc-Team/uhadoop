

# 产品价格

UHadoop产品包含的计费模式如下：
| 节点机型          | 计费说明  |
| ---------------- | -------------------- |
| 快杰型（O型）  | 按照CPU、内存、存储3个维度进行计费                   |
| GPU计算型 | 按配置节点、存储计费                   |
| 裸金属型、其他机型  | 按节点配置整体计费，其他机型（裸金属型除外）为历史机型，仅支持存量用户继续使用。|

> 说明：文档中的价格以华北（北京2）为例，其他地区价格可查看[UHadoop控制台](https://console.ucloud.cn/uhadoop/create)。
